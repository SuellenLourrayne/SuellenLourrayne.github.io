# Suellen Lourrayne - Personal Portfolio

Welcome to my personal portfolio! This website serves as a showcase of the projects I have worked on and highlights my skills in software development and programming. It's designed to give potential clients and employers a better understanding of my expertise and the type of work I can deliver.

## Features

- **Project Portfolio**: A collection of my completed projects, with descriptions and links to live demos or repositories.
- **Skills Section**: A detailed overview of my programming skills, technologies I've worked with, and tools I am proficient in.
- **Contact Information**: An easy way for visitors to get in touch with me for collaboration or job opportunities.

## How It Works

1. **Project Showcase**: Visitors can browse through various projects I've built, each featuring an overview, key technologies used, and links to the code or live version.
2. **Skills Overview**: A section that provides a clear outline of my technical abilities, including front-end, back-end, and full-stack development.
3. **Contact Form**: Visitors can fill out a contact form with their inquiries, which will be directed to my email or other preferred communication methods.

## Live Demo

You can view my portfolio website here: [Suellen Lourrayne - Personal Portfolio](https://suellenlourrayne.vercel.app/)

## Technologies Used

- **React.js**: For building the user interface and structuring the portfolio.
- **CSS**: For styling and creating a modern, responsive design.
- **GitHub Pages/Vercel**: For hosting and deploying the website.
